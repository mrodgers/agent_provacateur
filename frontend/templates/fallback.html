Usage Instructions

To use the Agent Provocateur system:

  1. Start the backend API server: ap-server --host 127.0.0.1 --port 8000
  2. Start the frontend server: python frontend/server.py --host 127.0.0.1 --port 3000
  3. Access the web UI in your browser: http://localhost:3000
